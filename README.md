# 🖥 We Care About Your Pet

Landing page desenvolvida com foco em apresentar serviços voltados ao cuidado de pets, com um design moderno, simples e responsivo.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 📄 Sobre o projeto

Este projeto consiste em uma landing page criada para destacar serviços de cuidado com animais de estimação, visando uma navegação clara e uma boa experiência para o usuário.
